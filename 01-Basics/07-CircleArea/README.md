# Circle Area Calculator

A simple Java program that calculates the area and circumference of a circle using a preset radius value. The program uses the built-in `Math.PI` constant for accurate calculations.

---

# Features

* **Area Calculation:** Calculates the area of a circle using the formula `π × radius²`.
* **Circumference Calculation:** Calculates the circumference using the formula `2 × π × radius`.

---

# Run

```bash
javac CircleArea.java
java CircleArea
```

The program outputs:

```text
Circle Calculator
Radius: 5.0
Area: 78.53981633974483
Circumference: 31.41592653589793
```
