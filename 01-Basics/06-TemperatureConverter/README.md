# Temperature Converter

A simple Java program that converts temperatures between Fahrenheit (F), Celsius (C), and Kelvin (K). The program uses preset values and displays the converted results without requiring user input.

---

# Features

* **Celsius Conversion:** Converts Celsius values into Fahrenheit and Kelvin.
* **Fahrenheit Conversion:** Converts Fahrenheit values into Celsius and Kelvin.
* **Kelvin Conversion:** Converts Kelvin values into Celsius and Fahrenheit.

---

# Run

```bash
javac TempConverter.java
java TempConverter
```

The program outputs:

```text
Temperature Converter
--------------------

Celsius: 25.0 C
Fahrenheit: 77.0 F
Kelvin: 298.15 K

Fahrenheit: 77.0 F
Celsius: 25.0 C
Kelvin: 298.15 K

Kelvin: 300.0 K
Celsius: 26.85 C
Fahrenheit: 80.33 F
```
