# Simple Calculator

A basic Java calculator program that performs addition, subtraction, multiplication, and division using preset numbers. 

---

# Features

* **Addition:** Adds two numbers together.
* **Subtraction:** Finds the difference between two numbers.
* **Multiplication:** Multiplies two numbers.
* **Division:** Divides one number by another.

---

# Run

```bash
javac Calc.java
java Calc
```

The program outputs:

```text
Simple Calculator
-----------------
Number 1: 20.0
Number 2: 5.0

Addition: 25.0
Subtraction: 15.0
Multiplication: 100.0
Division: 4.0
```
