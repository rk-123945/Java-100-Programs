# Simple Math

A Java program that uses arithmetic operations using two integer variables. It performs addition, subtraction, multiplication, division, and modulus, then displays the results.

---

# Features

* **Basic Arithmetic:** Performs addition, subtraction, multiplication, division, and modulus.
* **Easy to Understand:** Uses simple variables and `System.out.println()` statements.

---

# Run

```bash
javac SimpleMath.java
java SimpleMath
```

The program outputs:

```text
Number 1: 20
Number 2: 5
Addition: 25
Subtraction: 15
Multiplication: 100
Division: 4
Modulus: 0
```
