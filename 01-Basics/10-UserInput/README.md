# Student Grade Calculator

A Java program that collects student assignment grades, calculates the overall average, determines the final letter grade, checks pass/fail status, and provides personalized feedback.

---

# Features

* **Grade Collection:** Allows users to enter multiple assignment scores.
* **Input Validation:** Ensures grades are between 0 and 100.
* **Average Calculation:** Calculates the student's overall percentage.
* **Letter Grade System:** Converts averages into A, B, C, D, or F grades.
* **Pass/Fail Detection:** Determines whether the student passed the course.
* **Feedback:** Provides different messages based on performance.
* ** Java :** Uses loops, conditionals, `switch` statements, variables, and user input.

---

# Run

```bash
javac UserInput.java
java UserInput
```
