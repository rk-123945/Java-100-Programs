# Simple Interest Calculator

A Java program that calculates simple interest using preset values for the principal amount, interest rate, and time period. The program uses the standard simple interest formula and displays the calculated interest and total amount.

---

# Features

* **Simple Interest Calculation:** Calculates interest using the formula `(Principal × Rate × Time) / 100`.
* **Total Amount Calculation:** Adds the interest earned to the original principal.

---

# Run

```bash
javac SimpleInterest.java
java SimpleInterest
```

The program outputs:

```text
Simple Interest Calculator
-------------------------
Principal: $1000.0
Rate: 5.0%
Time: 3.0 years
Simple Interest: $150.0
Total Amount: $1150.0
```
