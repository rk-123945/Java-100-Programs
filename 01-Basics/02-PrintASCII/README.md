# Custom ASCII Cat Generator

Generates customizable ASCII cats using loops. You can adjust variables in the code to stretch the cat's body or extend its tail.

---

# Features

- **Customizable Height:** Use loops to make a regular cat or a long cat.
- **Extendable Tail:** Control how long the cat's tail is using standard Java loops.


---
# Run
javac PrintASCII.java
java PrintASCII

With `bodyHeight = 3` and `tailLength = 4`, the program outputs:
```
 /\_/\ 
( o.o )
 |   | 
 |   | 
 |   | 
(___.___)~~~~
```
