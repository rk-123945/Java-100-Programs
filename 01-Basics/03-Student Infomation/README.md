# Student Information Display

A Java program that stores and displays student information using a class, constructor, and object-oriented programming concepts.

---

# Features

* **Student Class:** Stores a student's name, age, course, and student ID.
* **Constructor:** Initializes student information when an object is created.
* **Display Method:** Prints the student's information in a clean, readable format.

---

# Run

```bash
javac Main.java
java Main
```

The program outputs:

```text
Student Information
-------------------
Name: John Doe
Age: 18
Course: Computer Science
Student ID: 1001
```
