# Swap Variables

A Java program that swaps the values of two variables using a temporary variable and displays the values before and after swapping.

---

# Features

* **Variable Swapping:** Exchanges the values of two integer variables.
* **Before and After Display:** Shows the variable values before and after the swap.

---

# Run

```bash
javac SwapVariables.java
java SwapVariables
```

The program outputs:

```text
Before Swapping:
First = 10
Second = 20

After Swapping:
First = 20
Second = 10
```
