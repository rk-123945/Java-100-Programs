# Average Calculator

A Java program that accepts numbers from the user until they type `done`, then calculates and displays the average of all numbers entered.

---

# Features

* **Continuous Input:** Accepts any number of numeric inputs from the user.
* **Done Command:** Stops accepting input when the user types `done`.
* **Average Calculation:** Computes and displays the average of all entered numbers.
* **Input Validation:** Notifies the user if an invalid value is entered.

---

# Run

```bash
javac AverageCalculator.java
java AverageCalculator
```

Example Run:

```text
Enter a number (or type 'done' to finish): 5
Enter a number (or type 'done' to finish): 10
Enter a number (or type 'done' to finish): 15
Enter a number (or type 'done' to finish): done

The average is 10.0
```
