# Tip Calculator

A Java program that calculates a restaurant tip, total bill, and the amount each person pays when splitting the bill.

---

# Features

* **Custom Tip Percentage:** Enter any tip percentage.
* **Bill Splitting:** Divide the total among multiple people.
* **Formatted Currency Output:** Displays values with two decimal places.

---

# Run

```bash
javac TipCalculator.java
java TipCalculator
```

Example Run:

```text
Restaurant Tip Calculator

Enter bill amount: $150
Enter tip percentage: 20
Enter number of people splitting the bill: 3

Bill Summary
Bill Amount: $150.00
Tip: $30.00
Total Bill: $180.00
Each Person Pays: $60.00
```
