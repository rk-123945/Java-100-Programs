# Speed Calculator
A Java program that calculates speed using user-provided distance and time values. The program converts time values when needed and also calculates average pace.

---

# Features
* **Speed Calculation:** Calculates speed using:speed = distance ÷ time.
* **Average Pace:** Calculates the amount of time required per distance unit.
* **User Input:** Accepts custom distance and time values.

---

# Run
```bash
javac SpeedCalculator.java
java SpeedCalculator
```
Example Run:
Speed Calculator
Enter distance traveled: 120
Enter time taken: 2
Enter distance unit (miles/km): miles
Enter time unit (hours/minutes): hours

Results
Distance: 120.0 miles
Time: 2.0 hours
Speed: 60.0 miles per hour
Average Pace: 0.016666666666666666 hours per miles
