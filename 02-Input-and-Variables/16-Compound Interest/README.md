# Compound Interest Calculator

A Java program that calculates compound interest using user-provided information. The program uses the standard compound
intrest formula and displayes the final balance and interest earned.  

---

# Features

* **Principal Input:** Enter the starting investment amount.
* **Custom Interest Rate:** Specify the annual percentage rate.
* **Compounding:** Choose how many times interest compounds each year.
* **Investment Duration:** Calculate growth over multiple years.
* **Interest Summary:** Displays both interest earned and final balance.

---

# Run

```bash
javac CompoundInterest.java
java CompoundInterest
```
