# Currency Converter

A Java program that converts money between several major currencies using user input and predefined exchange rates.

---

# Features

* **Multiple Currency Options:** Convert between USD, EUR, GBP, and JPY.
* **User Input:** Enter the amount and choose the desired conversion.
* **Formatted Output:** Displays values with two decimal places.
* **Menu System:** Uses a switch statement for selecting conversions.

---

# Run

```bash
javac CurrencyConverter.java
java CurrencyConverter
```

Example Run:

```text
Currency Converter
Enter amount to convert: 100

Choose the currency conversion:
1. USD to EUR
2. USD to GBP
3. USD to JPY
4. EUR to USD
5. GBP to USD
6. JPY to USD

Enter your choice: 1

Conversion Summary
Original Amount: 100.00 USD
Exchange Rate: 0.92
Converted Amount: 92.00 EUR
```
