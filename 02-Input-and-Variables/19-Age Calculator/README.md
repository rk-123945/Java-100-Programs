# Age Calculator

A Java program that calculates a person's current age using their birth date. The program uses Java's built-in date libraries to calculate years, months, days, and total days lived.

---

# Features
* **Age Calculation:** Calculates exact age using the user's birth date.
* **Results:** Displays age in years, months, and days

---

# Run
```bash
javac AgeCalculator.java
java AgeCalculator
```
Example Run:
Age Calculator
Enter birth year: 2005
Enter birth month (1-12): 6
Enter birth day: 15

Results
Birth Date: 2005-06-15
Current Date: 2026-07-14
Age: 21 years, 0 months, and 29 days
Total Days Lived: 7704
