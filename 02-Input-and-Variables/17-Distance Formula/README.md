# Distance Formula Calculator

A Java program that calculates the distance between two points on a coordinate plane using user-provided coordinates. The program also calculates the midpoint between the two points.

---

# Features
* **Distance Calculation:**Uses the distance formula: √((x2 - x1)² + (y2 - y1)²).
* **Midpoint Calculation:** Finds the center point between two coordinates.

---

# Run
```bash
javac DistanceFormula.java
java DistanceFormula
```

Example Run:
Distance Formula Calculator
Enter x1: 2
Enter y1: 3
Enter x2: 8
Enter y2: 11

Results
Distance: 10.0
Midpoint: (5.0, 7.0)
