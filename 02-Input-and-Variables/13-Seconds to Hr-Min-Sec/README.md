# Seconds to Hours Minutes Seconds Converter

A Java program that converts a user-entered number of seconds into hours, minutes, and seconds. The program also displays the converted time in `HH:MM:SS` format.

---

# Features

* **User Input:** Accepts the total number of seconds from the user.
* **Input Validation:** Ensures the entered value is a non-negative integer.
* **Time Conversion:** Converts seconds into hours, minutes, and remaining seconds.
* **Formatted Time:** Displays the result in `HH:MM:SS` format

---

# Run

```bash
javac TimeConverter.java
java TimeConverter
```

The program outputs:

```text
Seconds to Hours Minutes Seconds Converter
Enter total seconds: 7384

Time Conversion
Total Seconds: 7384
Hours: 2
Minutes: 3
Seconds: 4
Formatted Time: 02:03:04
```
